## Refresh

Exchange the refresh token for a fresh, rotated pair. Replaying a rotated token revokes
the chain.

<APIEndpoint
  method="POST"
  path="/v1/auth/refresh"
  title="Rotate the token pair"
/>

```bash
curl -X POST https://api.id.qeet.in/v1/auth/refresh \
  -H "Content-Type: application/json" \
  -d '{"refresh_token":"…"}'
```

## Current principal

<APIEndpoint
  method="GET"
  path="/v1/auth/me"
  title="Current user/tenant/session context"
/>

```bash
curl https://api.id.qeet.in/v1/auth/me \
  -H "Authorization: Bearer $ACCESS_TOKEN"
```
