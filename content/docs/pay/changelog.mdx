Qeet Pay is **in design**. There's no public API yet; this page will track shipped
milestones once development begins.

<Callout type="info" title="In design">
  Qeet Pay is documented from its product requirements — the repo has no active
  code yet. See [Concepts](/pay/concepts) for the planned model and the
  [roadmap](/pay/roadmap) for the phased plan.
</Callout>

## Design phase

- **Product requirements complete** — acceptance, payouts, subscriptions, invoicing,
  and India compliance (GST, UPI, NACH) defined.
- **Technical architecture drafted** — ledger model and integration surface. See
  [Concepts](/pay/concepts).

## Next

- First REST API v1 + OpenAPI spec. Tracked on the [roadmap](/pay/roadmap).
